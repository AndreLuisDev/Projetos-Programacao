HISTÓRIA

	- Criado décado 90 - Guido Van Rossum - Centro de Matemática Stichting na Holanda, sucessora da Linguagem ABC.
	-  linguagem ABC foi desenvolvida para o uso de não programadores, mas logo no início mostrou várias limitações e restrições , não agradando ao público. Guido começou a tarefa de criar uma linguagem de script simple, que possuísse algumas das melhores propriedades da ABC.
	- Atualmente, a linguagem Python é gerenciada pela Python Software Foundation. Possui uma licença de código aberto, chamada Python Software Foundation License, que é compatível com a General Public License (GNU) a partir da versão 2.1.1 e incompatível em certas versões anteriores.

|---------IDE-------|

	- Integrated Development Environment (IDE) ou Ambiente de Desenvolvimento Integrado:
	 	- software - combina ferramentas comuns para desenvolvimento em uma única interface gráfica
		- Isso facilita o desenvolvimento de aplicativos:
		- Possui Editor de código-fonte, Automação de compilação local e Debugger;

	- IDE PYCHARM
		- torna o desenvolvimento mais fácil, agradável. Aspecto moderno.
		- Recursos extras.
		- Instalar a versão Community que é gratuita para desenvolvedores.

		
	- Instale a IDE, clique em New environment using - Escolher a versão do interpretador que será usado no projeto (na versão que instalei não deu essa opção);
		- a pasta Virtualenv é o ambiente isolado do sistema operacional;
		- nela contém a versão do interpretador, a qual selecionamos na criação do projeto e seus módulos embutidos (builtins);
		- não marcar a opção de main.py (script para providenciar a criação do arquivo principal);
